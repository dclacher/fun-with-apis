Fun with APIs

In this web application, the user can search for a city and the system will provide information about the current weather conditions, as well as general information about the city. For the weather part, the application connects in the backend to the OpenWeatherMap API. To get general information about the city, it connects to the Wikipedia API.

User documentation (how to use it):

  • Go to this page.
  • Enter your search term (a city name) in the text box and click on the "Search" button.
  • If the APIs can find the city, the page will display information about the weather in there and some general information (a summary) from Wikipedia, with a link to the entire article.
  • If not found, an error should be displayed.
  • Hint: use the combination "<city name, country code>" for more accurate results. E.g.: Rome, IT

Final Note:

For the LOG entries/file containing the transactions with the APIs, please check the source codes uploaded with the assignment in Blackboard (ZIP file). The name of the log file is fun_with_api.log, in the root of the project structure.
